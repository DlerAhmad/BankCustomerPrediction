Bank Term-Deposit Prediction System

In this project we designed and implemented a bank deposit customer prediction system which is capable of predicting bank future customers based on the available data. Using the system, banks can save financial and human resources in their customer discovering process. The solution was able to predict the future customers with more than 85% accuracy.

How to run the program?

	-compile all java files using		javac *.java

	-run main program using			java MainProgram
	
	-follow instructions given by the program
